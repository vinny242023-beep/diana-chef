# Задача: Создать полный UI Style Guide на основе анализа существующей страницы

Ты выступаешь как Senior UI/UX Designer, Design System Architect и Frontend UI Analyst.

Твоя задача — провести максимально детальный визуальный и технический анализ страницы:

https://chefmaison.com/en-de/services/catering/

и на основании реального анализа сформировать полноценный **UI Style Guide / Design System Documentation**, позволяющий любому дизайнеру или фронтенд-разработчику создавать новые страницы в том же визуальном стиле без необходимости обращаться к исходному сайту.

---

# ВАЖНО

## Требования к анализу

- Используй только реальные данные, полученные из анализа страницы.
- Не придумывай значения.
- Если значение невозможно определить точно, укажи:
  - предполагаемое значение;
  - уровень уверенности (Высокий / Средний / Низкий).
- По возможности извлекай значения из CSS, Tailwind-классов, вычисленных стилей и структуры DOM.
- Для каждого элемента указывай:
  - визуальное описание;
  - точные параметры;
  - рекомендации по воспроизведению.

# Формат результата

Результат оформить как полноценный документ в Markdown.

Использовать иерархию заголовков:

```markdown
# Название раздела
## Подраздел
### Деталь
```

Все параметры оформлять в таблицах.

Пример:

| Свойство | Значение |
|-----------|-----------|
| Font Size | 16px |
| Weight | 400 |
| Line Height | 24px |
| Color | #111111 |
| Letter Spacing | 0px |
